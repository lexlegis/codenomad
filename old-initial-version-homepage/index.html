Learn to code

Code Nomad invites you to join an innovative program for learning skill necessary in a career of a successful software engineer. Attendants will be encouraged to work independently, but also to collaborate with the team to acheive a joint goal. The program is prepared for beginner level, and previous programing experience is not required.

Who is it for?

You want to become a programmer, but you have none (or almost none) previous experience in programming

Or you wonder if that would be the right job for you

You would like to know what it means to work on a real project

You want to learn but you don't know where would be the best to start from

If some of those are correct for you, this program will be right for you. Our main goal, beside teaching you a particular programming language, is to help you understand and develop other necessary assets for landing a software developer job - from learning about tools and practices that software developers use on a daily basis to productively complete their tasks, to developing a couple of projects that you can use to show and talk about at your first job interviews.
